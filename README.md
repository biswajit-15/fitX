# fitX 🏋️

A personal nutrition & fitness tracking app built with Flutter and Firebase.

## Features

- 📊 **Calorie tracking** — log daily calories with real-time progress ring
- 🥗 **Macro tracking** — protein, carbs and fat with visual progress bars
- 🔥 **Streak system** — stay motivated with daily login streaks
- 📅 **Date navigation** — browse and log meals for any date
- 🍱 **Meal planning** — weekly schedule and meal management
- 📷 **Barcode scanner** — add food instantly by scanning barcode
- 👤 **User profile** — personal targets and preferences
- 🔐 **Firebase auth** — secure login and signup

## Tech Stack

- **Frontend** — Flutter (Dart)
- **Backend** — Firebase Firestore
- **Auth** — Firebase Authentication
- **State management** — Provider

## Getting Started

### Prerequisites

- Flutter SDK `>=3.0.0`
- Dart SDK `>=3.0.0`
- Firebase project set up

### Installation

1. Clone the repo
```bash
   git clone https://github.com/biswajit-15/fitX.git
   cd fitX
```

2. Install dependencies
```bash
   flutter pub get
```

3. Add your Firebase config files
   android/app/google-services.json
   ios/Runner/GoogleService-Info.plist

4. Run the app
```bash
   flutter run
```
